##  Descripción del Proyecto

Este repositorio contiene las prácticas desarrolladas durante el taller de Python, enfocadas en fundamentos de programación, análisis de datos y visualización. El proyecto incluye notebooks de Jupyter con ejercicios sobre:

- Entrada y salida de datos
- Operaciones aritméticas y estructuras de control
- Manejo de listas y estructuras de datos
- Análisis estadístico con NumPy
- Visualización de datos con Matplotlib
- Manipulación de archivos CSV con Pandas

##  Instalación y Configuración

### 1. Clonar el Repositorio

```bash
git clone https://github.com/TU_USUARIO/TallerPython_MendezPadronGustavo.git
cd TallerPython_MendezPadronGustavo
```

### 2. Instalar Dependencias

Asegúrate de tener Python 3.8 o superior instalado. Luego instala las dependencias:

```bash
pip install -r requirements.txt
```

#### Dependencias Principales:
- **numpy** - Cálculo numérico y operaciones con arrays
- **pandas** - Análisis y manipulación de datos
- **matplotlib** - Visualización de datos
- **openpyxl** - Soporte para archivos Excel
- **jupyter** - Entorno de notebooks interactivos

##  Cómo Ejecutar el Código

### Opción 1: Ejecutar en Jupyter Notebook (Local)

1. Inicia Jupyter Notebook:
```bash
jupyter notebook
```

2. En el navegador, navega a la carpeta `notebooks/` y abre:
   - `TallerPython1_MendezPadronGustavo.ipynb`
   - `TallerPython2_MendezPadronGustavo.ipynb`

### Opción 2: Ejecutar en Google Colab

1. Sube los archivos `.ipynb` a tu Google Drive
2. Abre con Google Colab
3. Asegúrate de subir también los archivos de la carpeta `data/` si son necesarios

### Opción 3: Ejecutar Scripts Python

```bash
python src/example.py
```
